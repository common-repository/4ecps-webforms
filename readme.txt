The easy way to integrate the 4ECPS Webforms to your website.

== Description ==
For those people who have an active 4ECPS Web Form account, this plugin will enable you to publish web forms. To learn more about the 4ECPS Marketing Web Forms and to get your 4ECPS account, go to https://4ecps.com

Powerful Web Form Features

-WYSIWYG drag and drop visual form builder
-Conditional content
-Progressive profiling with dynamic content
-Full set of survey elements
-Dynamic storybook mode: Build an interactive dynamic story
-Custom fields
-Payment gateway (Stripe integration)
-Visual workflow builder
-Appointment scheduling calendar included
-Fully responsive


Unlike other online web form builder solutions that only deliver web forms, 4ECPS incorporates advanced marketing automation and phone call tracking capabilities to ensure you capture every prospect who views your form. Combine exceptional web forms with outstanding landing pages, and powerful phone tracking and you will see â€¦ there is no better solution than 4ECPS!


== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `4ECPS` and all its contents to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
